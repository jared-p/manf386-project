8// Declarations for the CODK_CpuReadWriteData class



9//



10// The read/write data access helper class serves as a wrapper to the 



11// output buffer passed into the Execute function. You can use the 



12// functions in this class to read and write STEP 7 data types to the 



13// output buffer. It takes the raw input from the ODK Module Execute 



14// function and provides structured access to the output data area.  



15// Multibyte values are converted from little endian to big endian formats 



16// as necessary.  C++ data types are converted to S7 data types in the write



17// operation.  This class provides write access and inherits read access 



18// (from CODK_CpuReadData) to the data area.



19//

52    //

58        {



59            return -1;  // Case of no write operations



60        }



61        return m_HighIndex;



62    }
