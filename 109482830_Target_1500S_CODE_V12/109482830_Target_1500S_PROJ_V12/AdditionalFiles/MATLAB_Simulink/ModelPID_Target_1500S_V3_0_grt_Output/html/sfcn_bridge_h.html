4 * File: sfcn_bridge.h     



5 *



6 * Abstract:



7 *   This file helps support a bridge between non-inlined S-functions



8 *   and the rtModel. When we did not have the notion of an rtModel,



9 *   non-inlined S-functions obtained model data by directly accessing



10 *   the model's SimStruct. With the introduction of the rtModel, 



11 *   this direct access is no longer possible. Therefore, we need to 



12 *   redefine some of the macros inside simstruc.h to instead access



13 *   fields of the rtModel. 



14 */
